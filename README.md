# autonomous-marketing-agency

A Fully Autonomous Content Marketing Agency that will input information about a brand such as brand guidelines and tone then pass that information to a wide range of different agents who have specialties in different things like copywriting for landing pages, SEO for blog writing, design for social media graphics, YouTube strategy for video scripting and content creation and more.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/autonomous-marketing-agency.git
cd autonomous-marketing-agency
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
